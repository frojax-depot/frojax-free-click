# FROJAX FREE CLICK — Flash más visible + sin scroll (100% pantalla)

- **Flash visual** mucho más notorio: anillo expansivo, brillo del círculo, LED intenso y **destello de pantalla** (más fuerte en el acento).
- Usa `100dvh` y `overflow:hidden` para que **no haya scroll** y el app **ocupe toda la pantalla** en cualquier teléfono.
- Controles compactados y responsivos para caber siempre.
- Mantiene: atajos (60–200) con estado activo, compás junto al BPM, modo entrenamiento (toggle-only), volumen/sonidos/acentos/compás variable.

Publicación:
1) Sube todo al root del repo.
2) Settings → Pages → Source: main + / (root)
3) Abre tu URL de GitHub Pages.
